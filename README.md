# Discord AI Image Generator Bot

A Discord bot that generates AI images using Stable Diffusion.

## Features

- AI image generation
- Discord slash command `/generate`
- Stable Diffusion API backend
- Simple setup

## Installation

Clone the repository

git clone https://github.com/yourname/discord-ai-image-bot.git

Enter project

cd discord-ai-image-bot

Run setup

bash scripts/setup.sh

Configure `.env`

DISCORD_TOKEN=your_bot_token
API_URL=https://your-api-url/generate

Run bot

python bot.py

## Command

/generate prompt:<description>

Example

/generate prompt: cyberpunk city at night
